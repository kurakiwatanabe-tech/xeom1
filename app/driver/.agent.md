Use this agent context when working in `app/driver`.

Focus areas:
- Flutter login and home flow for drivers.
- Firebase Cloud Messaging token registration on login.
- Driver backend API endpoints under `/api/drivers`.
- Android `google-services.json` setup and native Gradle plugin configuration.
- Sending FCM token to backend via `POST /api/drivers/:id/fcm-token` after authentication.
